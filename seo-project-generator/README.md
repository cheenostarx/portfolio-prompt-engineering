# Automated SEO Blog Post Generator

**Project Overview**: Multi-step prompt chain that generates high-quality, SEO-optimized blog posts from a topic and keywords.

## Problem
Content teams spend too much time on research and first drafts. Basic prompts produce generic, poorly structured content.

## Solution
A chained workflow: Research → Outline → Draft → SEO Optimization.

## Prompt Strategy & Iteration
**Version 1**: Single long prompt → Poor structure and weak SEO.  
**Version 2**: Split into chained steps.  
**Final Version**: Added readability targets and natural keyword integration.
